## Instructions for Build and Use

Steps to build and/or run the software:

1. Install Python 3.x on the system and verify it is available from the command line.
2. Download or clone the project files and place them in a local directory.
3. Open a terminal or command prompt in the project directory.
4. Run the program using the command: python main.py
5. The program will automatically create the movies.db database file and required tables if they do not already exist.

Instructions for using the software:

1. When the program starts, a menu of options is displayed in the terminal.
2. Choose an option by typing the corresponding number and pressing Enter.
3. Use Add a movie to insert new movie records into the database.
4. Use List all movies or Search movies by title to query and view stored data.
5. Use Update a movie to modify ratings, notes, or watch dates.
6. Use Delete a movie to remove a movie record from the database.
7. Use List movies by date range to filter movies by watched dates.
8. Use Show statistics to view aggregate information such as counts and average ratings.
9. Select Exit to safely close the application.
